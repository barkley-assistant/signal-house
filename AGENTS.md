# Agent instructions

This repository is worked on by AI coding agents.

Before making changes, read the relevant project docs and skill files.

Ensure to keep the `README.md` and `.env.example` up to date.

## Required reading

For any task:

* Read the relevant files in `docs/` before starting.
* Update any files in `docs/` on task completion. 

For frontend/UI work:

* Use `frontend` skill.

For backend/API/database work:

* Use `backend` skill.

For tasks that touch both frontend and backend:

* Read both `skills/frontend/*.md` and `skills/backend/*.md`.

## Working style

* Follow the existing project patterns.
* Keep changes scoped to the task.
* Avoid unnecessary rewrites.
